# Dataset Research Reference

Use this skill to prepare grounded source material for explicit APO datasets from:

- official datasets and benchmarks from accountable maintainers
- task-specific documentation, dataset cards, and labeling guides
- standalone mapping notes that capture how approved source fields can become eval rows, expected outputs, optional files, and assertions

Prefer official primary sources. Use secondary material only to discover or verify the primary source, and record why weaker sources were rejected.
